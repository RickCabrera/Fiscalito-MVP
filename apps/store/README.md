# Fiscalito Store

Marketplace web de servicios inteligentes para PYMEs y personas fisicas mexicanas. Cada servicio resuelve una obligacion legal o financiera (fiscal, laboral, contable).

## Tech Stack

- **Frontend**: React 19 + Vite + TypeScript
- **Auth**: Firebase Auth (email + Google)
- **DB**: Firestore
- **Backend fiscal**: Fiscal Agent API (FastAPI/Python)
- **LLM**: OpenAI / Anthropic

## Inicio rapido

```bash
npm install
npm run dev
```
